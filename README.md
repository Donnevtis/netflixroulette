# NETFLIXROULETTE

**A single page application, which will allow users to search the Movies DB database**

## Project setup

### Install

```bash
npm ci
```

### Install and start server

```bash
cd backend
npm install
npm start
```

### Compiles and hot-reloads for development

```bash
npm start
```

### Lint project

```bash
npm run lint
```

### Compiles and minifies frontend for production

```bash
npm run build
```
